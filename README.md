# Ping-pong
Прототип игры "Пинг-понг"

В репозитории лежит прототип игры "Пинг-понг". Это мультиплеерная игра, расчитанная на 2 игрока. 
Главная задача: не дать мячу выйти за левую и правую рамки игрового окна (от верхней и нижней он отскакивает, не влияя на исход игры). Если это произошло, то счётчик над ракеткой, пропустившей снаряд, увеличится на единицу. Когда счётчик достигает значения "5", игра завершается, а на экране всплывает надпись с обозначением проигравшего оппонента. 

Игра реализована на языке Python с помощью библиотеки Pygame.

Управление левой ракеткой осуществляется при помощи клавиш клавиатуры "W" и "S", правой - "стрелка вверх" и "стрелка вниз". Движение происходит по маршруту вверх-вниз и наоборот, в зависимости о нажатых кнопок. Мяч движется автоматически и со временем ускоряется, усложняя игру.
Игру можно завершить, если нажать на крестик в правом верхнем углу окна приложения.
